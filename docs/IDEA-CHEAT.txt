
*** TO CREATE NEW PROJECT

1) IMPORT from Existing Source

2) Choose option for SBT Template

3) Choose option for Auto-Import

4) Choose option for JDK1.8

5) ENTER through all other DEFAULTs!

6) Rclick on RHS SBT and hit REFRESH icon on top LHS

*** TO DELETE PROJECT

1) CD to project directory where build.sbt is

2) rm -rf .idea

3) rm -rf target

4) rm *.iml
